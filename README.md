# 2026-2030 Technician Question Pool with ecfr.org link

## Prerequisites

* uv
* LibreOffice


## Installation & Setup


1. Clone the repository:

```bash
git clone https://github.com/mlouielu/2026-2030-Technician-Question-Pool-with-ecfr.org-link
cd 2026-2030-Technician-Question-Pool-with-ecfr.org-link
```

2. Create venv

```bash
uv sync
```


## Generate the linked PDF

```bash
bash ./run.sh
```
